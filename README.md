RAILS Consulting - Sitio web (Vite + React + Tailwind)

Contenido:
- Proyecto listo para desarrollo con Vite.
- Logo en /assets/logo.png (extraído del PDF que subiste).
- Dominio: rails-consulting.com
- Correo principal configurado: tsi@rails-consulting.com

Instrucciones rápidas para desplegar (en tu computadora):

1) Instalar dependencias:
   npm install

2) Levantar en modo desarrollo:
   npm run dev

3) Construir:
   npm run build

4) Para desplegar en Vercel:
   - Crear cuenta en vercel.com (puedes usar tu GitHub)
   - Importar este proyecto desde el repositorio (o subir los archivos)
   - En Vercel, en Settings -> Domains, agregar: rails-consulting.com
   - Vercel detectará la app y desplegará automáticamente.

Si quieres, yo lo despliego por ti en Vercel y conectamos el dominio (yo hago todo sin que toques nada).
